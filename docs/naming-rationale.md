# 命名与品牌考量

## 和 OpenClaw 的呼应

现在主产品叫 OpenClaw，配套执行端叫 ClawMe，同一套「Claw」词根，容易记、也容易形成产品线：

- **OpenClaw** = 大脑  
- **ClawMe** = 为你出手的那只手  

若叫 MoltMe，和 OpenClaw 没有明显词根联系，品牌会散一点。

## 小结

- **Claw** = 爪、抓、动手 → 适合「执行端/桥梁」。
- **Molt** = 蜕皮、进化 → 适合「会成长的 AI」，不太适合「替我操作设备」的这款产品。

更推荐继续用 **ClawMe**（中文 虾me 保留），和 OpenClaw 一致，语义也最贴。Molt 可以留在 Moltbot/历史品牌或别的「进化/蜕变」向产品里用。

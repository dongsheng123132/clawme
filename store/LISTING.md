# Chrome Web Store Listing Content

## Extension Name
ClawMe - AI Agent Butler for Your Browser

## Short Description (max 132 characters)
AI Agent sends instructions, you confirm, browser executes. Fill forms, tweet, email, click — your AI butler in the side panel.

## Detailed Description

ClawMe is the bridge between AI Agents and your browser. Your AI Agent thinks and decides — ClawMe executes for you, with your permission.

🦞 HOW IT WORKS
1. Your AI Agent (OpenClaw, or any HTTP-capable agent) sends a structured instruction to ClawMe
2. ClawMe shows it in the browser side panel with a preview
3. You review and click "Execute" — the browser carries out the action
4. Semi-automatic: you stay in control, AI helps you be productive

📋 SUPPORTED INSTRUCTIONS (7 types)
• remind — Show a desktop notification with title and message
• open_url — Open any URL in a new tab
• compose_tweet — Open Twitter/X with pre-filled tweet text
• compose_email — Open Gmail with pre-filled recipient, subject, and body
• fill_form — Auto-fill form fields on any website (supports rich text editors)
• click — Click any element on a page by CSS selector
• extract — Extract text or attributes from a page and return to the agent

⚡ KEY FEATURES
• Side Panel UI — Always accessible, no popup needed
• Auto-polling — Checks for new instructions every 30 seconds
• Desktop notifications — Never miss an instruction from your AI
• Multi-step Workflows — Chain instructions with progress visualization
• Execution log — Track what was executed and results
• Auto-execute mode — Skip confirmation for trusted agents
• Dark mode — Follows your system preference
• Self-hosted backend — Your data stays on your server

🔧 SETUP
1. Install the extension
2. Run the ClawMe backend on your server (or locally)
3. Enter your Backend URL and Token in the side panel settings
4. Connect your AI Agent — done!

Backend: https://github.com/dongsheng123132/clawme
Docs: https://clawme.net

🔒 PRIVACY
• No data collection, no analytics, no tracking
• All communication is between your agent, your backend, and your browser
• Open source (AGPL-3.0) — audit the code yourself
• Self-hosted: you control everything

🌍 WORKS WITH
• OpenClaw (built-in plugin)
• Any AI Agent that can make HTTP POST requests
• Any self-hosted or cloud backend

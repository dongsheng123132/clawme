为什么用 ClawMe

🤖

Agent 指令直达设备

不再复制粘贴 — Agent 发指令，你的设备直接执行：发推、写邮件、填表单、点击、抓取。

📱

跨平台

浏览器插件 + 手机 PWA，一个 Token 连接所有设备。扫码即连，无需复杂配置。

🔒

你说了算

半自动执行：每条指令你确认后再执行，也可开启自动模式。Token 授权，只认你的 Agent。

🌐

开源 AGPL

代码完全开源，社区共建。自部署或用托管服务，你的数据你做主。

⚡

多步骤 Workflow

一组指令按顺序自动执行，支持进度可视化和中止。适合复杂自动化场景。

🔗

任意 Agent 接入

不绑定特定 Agent — OpenClaw、自研 Agent 或任何支持 HTTP 的 AI 都可接入。

三步连通

1

安装客户端

Chrome 安装浏览器插件，或手机打开本页添加到主屏幕（PWA）。

2

扫码连接

用手机扫描连接二维码，自动配置 Backend 地址和 Token，一秒连通。

3

Agent 发指令，你执行

OpenClaw 等 Agent 发送指令 → 你的设备收到通知 → 点击执行或自动完成。
